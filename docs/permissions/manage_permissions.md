High-level `allow_manage_permissions` permission includes the following
low-level permissions:
- `allow_permission_create`
- `allow_permission_delete`


- `allow_permission_name_read`
- `allow_permission_description_read`
- `allow_permission_value_read`


- `allow_permission_name_update`
- `allow_permission_description_update`
- `allow_permission_value_update`
