#### Diagrams
1. All the diagrams in this assignment are made using 
[draw.io](https://www.drawio.com)
([`diagrams.net`](https://app.diagrams.net)) tool.
2. To open existing diagram you need either:
   - Get a link from teammate with edit permissions, open it.
   - Download a `.drawio.xml` file, go to `diagrams.net`, then click
`File` >> `Import from` >> `Device...` >> select your file and open.
All changes you add to the diagram this way must be commited to
the GitHub repository.
